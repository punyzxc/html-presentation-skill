# 🎨 HTML Presentation Skill

Персональный скилл для GitHub Copilot CLI — автоматическое создание профессиональных HTML-презентаций с glassmorphism дизайном.

## 🚀 Быстрая установка

### Вариант 1: Глобальный скилл (работает везде)
```bash
# Клонируй в персональную папку Copilot
git clone https://github.com/YOUR_USERNAME/html-presentation-skill.git ~/.copilot/skills/html-presentation
```

### Вариант 2: Для конкретного проекта
```bash
mkdir -p .github/skills
git clone https://github.com/YOUR_USERNAME/html-presentation-skill.git .github/skills/html-presentation
```

## 📋 Использование

Просто скажи Copilot:
```
Создай презентацию на тему "Искусственный интеллект"
```

Или:
- "Сделай презу про машинное обучение"
- "Make presentation about UX design"
- "Нужны слайды на тему экологии, 10 штук"

## ✨ Возможности

- 🎨 **Glassmorphism дизайн** — современный стеклянный эффект
- ✨ **Плавные анимации** — fade-in, scale, stagger
- 🌈 **7 цветовых тем** — автоподбор под контент
- 🧭 **Полная навигация** — стрелки, клики, точки, прогресс
- 📱 **Адаптивность** — работает на любых экранах
- 🎯 **Один HTML-файл** — никаких зависимостей

## 📁 Структура репозитория

```
html-presentation-skill/
├── SKILL.md          # Основные инструкции для Copilot
├── README.md         # Этот файл
└── full-template.md  # Полные CSS/JS шаблоны
```

## 🔧 Что создаётся

Один HTML-файл с:
- Встроенными CSS стилями
- Встроенным JavaScript
- Фоновыми анимированными орбами
- Точками навигации
- Прогресс-баром
- Счётчиком слайдов
- Keyboard navigation (←→, пробел)
- Scroll-snap для плавного скролла

## 📝 Типы слайдов

1. **Title Slide** — заголовок, подзаголовок, автор
2. **Content Slide** — текст и списки
3. **Grid Slide** — сетка карточек (2-4 колонки)
4. **Stats Slide** — крупные числа
5. **Quote Slide** — цитата
6. **Conclusion Slide** — выводы
7. **Thank You Slide** — финал

## 🛠 Требования

- GitHub Copilot CLI
- Любой современный браузер
