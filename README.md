# 🎨 HTML Presentation Skill

Персональный скилл для GitHub Copilot CLI — автоматическое создание профессиональных HTML-презентаций с glassmorphism дизайном.

## 🚀 Быстрая установка

### Вариант 1: Глобальный скилл (работает везде)
```bash
# Клонируй в персональную папку Copilot
git clone https://github.com/YOUR_USERNAME/html-presentation-skill.git ~/.copilot/skills/html-presentation
```

### Вариант 2: Для конкретного проекта
```bash
mkdir -p .github/skills
git clone https://github.com/YOUR_USERNAME/html-presentation-skill.git .github/skills/html-presentation
```

## 📋 Использование

Просто скажи Copilot:
```
Создай презентацию на тему "Искусственный интеллект"
```

Или:
- "Сделай презу про машинное обучение"
- "Make presentation about UX design"
- "Нужны слайды на тему экологии, 10 штук"

## 🔧 Что создаётся

Один HTML-файл с:
- Встроенными CSS стилями
- Встроенным JavaScript
- Фоновыми анимированными орбами
- Точками навигации
- Прогресс-баром
- Счётчиком слайдов
- Keyboard navigation (←→, пробел)
- Scroll-snap для плавного скролла

## 📝 Типы слайдов

1. **Title Slide** — заголовок, подзаголовок, автор
2. **Content Slide** — текст в glass-card контейнере
3. **Grid Slide** — сетка карточек (2-4 колонки)
4. **Stats Slide** — крупные числа и статистика
5. **Quote Slide** — цитата в стилизованном боксе
6. **Timeline Slide** — хронология и процессы
7. **Feature Slide** — иконка + текст в блоке
8. **Conclusion Slide** — выводы
9. **Thank You Slide** — финал

**Ключевое правило:** Все текстовые элементы находятся в специальных контейнерах для профессионального вида.

## 📚 Документация

### Главное правило: НЕ "голого текста"!

Ваш текст **ВСЕГДА** должен быть в одном из контейнеров:
- `.glass-card` — основной текст
- `.grid-2 / .grid-3 / .grid-4` — сравнения, статистика
- `.timeline-content` — процессы
- `.key-point` — выделение
- `.quote-box` — цитаты
- `.feature-block` — иконка + текст
- `.highlight-box` — статус/важное

2. **[SVG_ICONS_GUIDE.md](docs/SVG_ICONS_GUIDE.md)**
   - 30+ готовых профессиональных SVG иконок
   - Организация по категориям
   - Почему SVG лучше эмодзи
   - Копируй-готовые примеры

3. **[BEST_PRACTICES.md](docs/BEST_PRACTICES.md)**
   - Чеклист идеального слайда
   - Рекомендации по структуре
   - Частые ошибки и их исправления
   - Примеры полных слайдов
