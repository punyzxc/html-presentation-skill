# 📚 Руководство по поиску и верификации информации для презентаций

> **Цель:** Обеспечить достоверность, актуальность и научность всех данных в презентациях.  
> **Результат:** Выступления без страха "покраснеть" перед фактологическими замечаниями.

---

## 🎯 Система 4 уровней доверия

### Уровень 1️⃣ — НАДЕЖНЫЕ (Green) 🟢
**Используй в 99% случаев. Факты подтверждены множеством источников.**

| Источник | Примеры | Когда использовать | Типичная задержка обновления |
|----------|---------|-------------------|------------------------------|
| **Научные журналы** | Nature, Science, IEEE, ACM | Технологии, исследования | 1-2 года |
| **Статистика ООН/Всемирного банка** | UN Data, World Bank Open Data | Экономика, демография, развитие | 1-2 года |
| **Правительственные агентства** | NASA, NOAA, NIH, Министерства РК | Официальные данные, политика | 6 мес - 1 год |
| **Авторитетные СМИ** | Reuters, BBC, AP News, BBC | Новости, исторические события | До часа |
| **Национальные библиотеки** | Library of Congress, Национальная библиотека РК | История, культура, литература | 5+ лет (статично) |
| **Образовательные платформы** | Coursera (от вузов), MIT OpenCourseWare, edX | Научный контент, образование | 1-2 года |
| **GitHub/официальные реп** | Tensorflow, React, Linux repo | Технологии, версии ПО | В реальном времени |
| **Википедия (для истории/основ)** | Wikipedia (история, биографии) | Исторические факты, определения | Месяцы |

**Как проверить:**
```
✅ Есть автор и дата публикации?
✅ Есть источники/ссылки в статье?
✅ Это организация с авторитетом?
✅ Информация подтверждена 2+ источниками?
```

---

### Уровень 2️⃣ — ХОРОШИЕ (Yellow) 🟡
**Используй с проверкой. Данные обоснованны, но требуют перекрестной проверки.**

| Источник | Примеры | Когда использовать | ⚠️ Осторожность |
|----------|---------|-------------------|--------------------|
| **Издания среднего рейтинга** | Forbes, TechCrunch, Habr, Médium | Новости, тренды, мнения | Проверить источники |
| **Компании (официальные блоги)** | Google Blog, Microsoft Research, Apple Newsroom | Анонсы продуктов, официальная позиция | Маркетинг-окрас |
| **Статьи в medium.com** | Medium (от экспертов) | Инсайты, рассказы опыта | Проверить автора |
| **Отчёты консультанг компаний** | Gartner, IDC, McKinsey (публичные) | Рыночные прогнозы, анализ | Дорогие → платные данные |
| **YouTube от экспертов** | Veritasium, 3Blue1Brown, CrashCourse | Объяснения сложных концепций | Проверить факты в описании |
| **Подкасты авторитетных авторов** | Joe Rogan (с экспертами), TED Talks | Мнения, интервью | Субъективный взгляд |
| **Reddit threads (от проф)** | r/science, r/AskScience | Коммьюнити-знание | Может быть ошибка |

**Как проверить:**
```
✅ Автор указан и имеет опыт?
✅ Есть ссылки на первоисточники?
✅ Нашёл подтверждение в Level 1?
✅ Дата актуальна (не старше 2-3 лет)?
```

---

### Уровень 3️⃣ — ОСТОРОЖНО (Orange) 🟠
**Используй ТОЛЬКО если Level 1-2 недоступны. Требуется множественная проверка.**

| Источник | Примеры | Когда использовать | 🚩 Красные флаги |
|----------|---------|-------------------|-------------------|
| **LinkedIn посты** | LinkedIn (мнения, инсайты) | Профессиональные перспективы | Не проверено, только мнение |
| **Twitter/X** | Twitter (от экспертов) | Актуальные новости | Очень субъективно |
| **Презентации на конф** | SlideShare, Speaker Deck | Инсайты выступающих | Может быть реклама |
| **Неофициальные источники** | Блоги энтузиастов, форумы | Нишевые темы | Личная интерпретация |
| **Старые статьи** | Википедия (в некоторых темах), 2010+ | Исторический контекст | Может быть устарело |

**Как проверить:**
```
⚠️ Обязательно найти Level 1-2 подтверждение!
⚠️ Упомянуть в презентации: "По данным..." или "Согласно..."
⚠️ Быть готовым к возражениям
```

---

### Уровень 4️⃣ — НЕ ИСПОЛЬЗОВАТЬ (Red) 🔴
**Избегай полностью. Источники недостоверные, враждебные, пропагандистские.**

| 🚫 Что НЕ использовать | Почему | Альтернатива |
|------------------------|--------|-------------|
| Сплетни, слухи | Без подтверждения | Найти официальное подтверждение |
| Сайты с очевидной ложью | Дезинформация | Проверить через factcheck.org |
| Государственные СМИ враждебных стран | Пропаганда | Использовать независимые источники |
| Паранаука, лженаука | Противоречит науке | Консультировать учёных |
| Заплаченный контент без пометки | Скрытая реклама | Искать беспристрастные источники |
| Статьи без автора и даты | Невозможна проверка | Требовать прозрачность |

## 🔍 Пошаговый процесс верификации информации

### Шаг 1: Определи ТИП информации

```
Категория данных:
┌─ ФАКТЫ (числа, даты, события)
│  └─ Требуют конкретных источников
├─ СТАТИСТИКА (проценты, тренды)
│  └─ Требуют методологии и периода
├─ МНЕНИЯ (анализ, рекомендации)
│  └─ Требуют авторитета автора
├─ ПРОГНОЗЫ (будущее, гипотезы)
│  └─ Требуют обоснования "почему"
└─ ИСТОРИЧЕСКИЕ СОБЫТИЯ
   └─ Требуют нескольких независимых источников
```

**Пример:**
- ❌ "В 2020 году облачные вычисления выросли на 37%" — факт, нужен источник
- ✅ "По отчёту IDC, облачные вычисления выросли на 37% в 2020 году" — есть источник

---

### Шаг 2: ПОИСК информации

#### 2.1 Для технологий:
```
1. GitHub issues/commits (официальная документация)
2. Papers with Code (научные работы)
3. ArXiv.org (препринты, новейшие исследования)
4. Официальные документации проектов
5. IEEE Xplore (научные статьи)
6. Google Scholar (поиск по научным работам)
```

**Пример поиска:**
```
❌ "Как работает машинное обучение?"
✅ "site:arxiv.org machine learning 2024"
✅ "site:github.com tensorflow documentation"
```

#### 2.2 Для статистики и экономики:
```
1. World Bank Open Data (worldbank.org)
2. UN Data (unstats.un.org)
3. OECD Statistics (oecd.org)
4. Национальные статистические агентства (Росстат, BLS)
5. Gartner/IDC (платно, но авторитетно)
6. IMF World Economic Outlook
```

#### 2.3 Для истории и культуры:
```
1. Library of Congress (loc.gov)
2. Википедия (история, биографии)
3. Архивы национальных библиотек
4. Academic databases (JSTOR, ProQuest)
5. Britannica Encyclopedia
6. Primary sources (оригинальные документы)
```

#### 2.4 Для новостей и текущих событий:
```
1. Reuters (новости, исследования)
2. AP News (American Press)
3. BBC (британское ТВ)
4. Bloomberg (экономика, финансы)
5. Associated Press (проверенные факты)
6. fact-checkers (factcheck.org, snopes.com)
```

---

### Шаг 3: ПРОВЕРКА через "Треугольник доверия"

**Правило 3 источников:**
```
                    Источник 1
                       ▲
                      / \
                     /   \
                    /     \
                   /       \
                  /         \
            Источник 2 — Источник 3

✅ Если все 3 совпадают → факт достоверен
⚠️ Если 2 из 3 совпадают → нужна осторожность
❌ Если все разные → нужна переверка
```

**Практический пример:**
```
Информация: "5.3 миллиарда человек используют интернет"

Проверка:
1️⃣ UN World Telecommunications — 5.3B (2023)
2️⃣ Statista Report — 5.3B (2023)
3️⃣ We Are Social — 5.3B (2023)

✅ ФАКТ ПОДТВЕРЖДЁН. Можно использовать.
```

---

### Шаг 4: ЧЕКА-ЛИСТ перед добавлением в слайд

```
□ Источник найден и указан?
□ Источник входит в Level 1-2?
□ Дата актуальна (не старше 3-5 лет для быстрых темы)?
□ Информация подтверждена 2+ независимыми источниками?
□ Цифры, проценты имеют методологию?
□ Есть ссылка на оригинальный источник?
□ Я готов оправдать эту информацию, если спросят?
□ Язык чётких, без преувеличений и манипуляций?
```

## 📋 Категории информации: рекомендации по глубине проверки

| Категория | Уровень проверки | Минимум источников | Макс. возраст данных |
|-----------|------------------|-------------------|----------------------|
| **История** | Средняя | 2-3 | 50+ лет (статично) |
| **Технологии** | Высокая | 2-3 | 1 год |
| **Статистика** | Высокая | 3+ | 2-3 года |
| **Цитаты** | Очень высокая | Найти оригинал | Любой |
| **Прогнозы** | Средняя | 2 | Дата прогноза важна |
| **Определения** | Низкая | 1 | Лет 5-10 |
| **Новости** | Высокая | 3+ | Месяц |
| **Личные истории** | Очень высокая | Оригинал | Как есть |

## 🛠️ Инструменты для проверки фактов

### Автоматические fact-checkers:
```
1. Google Fact Check (https://toolbox.google.com/factcheck)
   └─ Проверяет заявления против известных проверок

2. Snopes.com (https://www.snopes.com/)
   └─ Специализируется на интернет-слухах

3. FactCheck.org (https://www.factcheck.org/)
   └─ Проверяет политические и социальные заявления

4. Full Fact (https://fullfact.org/)
   └─ Британский fact-checker, очень качественный

5. PolitiFact (https://www.politifact.com/)
   └─ Для политической информации

6. Retraction Watch (https://retractionwatch.com/)
   └─ Отследить опровергнутые научные работы
```

### Поисковые операторы для проверки:
```bash
# Найти оригинальный источник
site:scholar.google.com "ваша цитата"
site:arxiv.org "ваш термин"

# Проверить дату публикации
"информация" "2024" site:bbc.com

# Найти противоречивые данные
"информация" "NOT" site:factcheck.org

# Поиск только в научных работах
"ваш запрос" filetype:pdf

# Последние новости
"информация" before:2024-01-01
```

## 🔐 Признаки ненадёжного источника

### 🚩 Красные флаги:
```
❌ Нет автора или дата публикации неясна
❌ Нет ссылок на источники информации
❌ Сенсационный заголовок с всеми заглавными буквами
❌ Множество орфографических ошибок
❌ Выраженная политическая/религиозная предвзятость
❌ "Клик сюда для подробнее" вместо фактической информации
❌ Выглядит как спам или скам
❌ Использует только анекдотические доказательства
❌ Не обновлялась годами
❌ Цитирует самого себя в качестве источника
```

### ✅ Признаки надёжного источника:
```
✅ Ясный автор с биографией/квалификацией
✅ Точная дата публикации и последнего обновления
✅ Ссылки на первоисточники и академические работы
✅ Нейтральный, объективный язык
✅ Признаёт противоположные взгляды
✅ Регулярно обновляется
✅ Есть раздел "источники" или "библиография"
✅ Работает за счёт подписок или государства (не паразитит на кликах)
✅ Имеет редакционную политику
```

## 📝 Документирование источников

### Для каждого слайда с фактом добавляй:

```html
<!-- Пример кода в презентации -->
<p>
  Согласно <strong>отчёту McKinsey (2024)</strong>, 
  использование ИИ выросло на <strong>45%</strong> в корпоративном секторе.
  
  <!-- Сноска или ссылка -->
  <a href="[URL отчёта]" class="source">📌 Источник</a>
</p>
```

### CSS для обозначения источников:
```css
.source {
  font-size: 0.8em;
  color: var(--accent-primary);
  text-decoration: none;
  cursor: pointer;
}

.source:hover {
  text-decoration: underline;
}

.source::before {
  content: " [";
}

.source::after {
  content: "]";
}
```

## 🎓 Шпаргалка: Что говорить если спросят на выступлении

```
Q: "Откуда вы это взяли?"
A: "По отчёту [Организация] от [год], 
    опубликованному на [источник]. 
    Я проверил эту информацию по трём независимым источникам."

Q: "Это данные актуальны?"
A: "Да, это [год]. Если вы найдёте более свежие данные, 
    я с удовольствием их рассмотрю после выступления."

Q: "Я где-то видел другую цифру"
A: "Возможно, речь идёт о разных периодах или методологиях. 
    Я использую [конкретный источник]. 
    Предлагаю сравнить после презентации."

Q: "Это не совпадает с моим опытом"
A: "Спасибо за замечание. Статистика — это усредненные данные, 
    в реальности могут быть исключения. 
    Было бы интересно услышать ваш опыт подробнее."
```

- Какой факт неправильный?
